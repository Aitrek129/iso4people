Как записать образ на флешку

Важно!

⚠️ Все данные на флешке будут удалены. Перед началом скопируйте важные файлы в другое место.

Windows

Способ 1: Rufus

  1. Скачайте и запустите Rufus.
  2. Вставьте флешку, выберите её в поле «Устройство».
  3. Нажмите «ВЫБРАТЬ» и укажите скачанный ISO-файл.
  4. Остальные параметры оставьте по умолчанию, нажмите «СТАРТ».
  5. Дождитесь завершения записи.

Способ 2: BalenaEtcher

  1. Скачайте BalenaEtcher и установите.
  2. Выберите ISO-файл, выберите флешку, нажмите «Flash!».

Linux

Графический способ (рекомендуется)

  1. Откройте Меню
  2. В строке поиска напишите "Запись образа на USB-накопитель"
  3. Запустите приложение
  4. В поле ISO-образ выберите ранее скачанный файл
  5. В поле USB-накопитель выберите свою флешку
  6. Нажмите "Записать"

Командная строка (dd)

Откройте терминал и выполните:

sudo dd if=/путь/к/образу.iso of=/dev/sdX bs=4M status=progress && sync

Замените /dev/sdX на вашу флешку (например, /dev/sdb). Будьте внимательны: не ошибитесь с устройством!

macOS

BalenaEtcher

Скачайте версию для macOS, запустите и выберите ISO и флешку.

Командная строка (dd)

Сначала определите диск флешки: diskutil list. Затем размонтируйте: diskutil unmountDisk /dev/diskX. Запишите:

sudo dd if=/путь/к/образу.iso of=/dev/rdiskX bs=1m

Используйте rdiskX для ускорения. После завершения выполните diskutil eject /dev/diskX.
